# arviseg

Astro 6.x static site with Tailwind CSS 4.x. Basic corporate website.

## Commands
- `npm run dev` - Start dev server at localhost:4321
- `npm run build` - Build to ./dist
- `npm run preview` - Preview built site

## Setup
- Node >= 22.12.0 required

## Notes
- No tests, linting, or typechecking configured (vanilla Astro starter)
- Tailwind 4.x uses Vite plugin (`@tailwindcss/vite`), not PostCSS config
- Static site with `.astro` pages in `src/pages/`
